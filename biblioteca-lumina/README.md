# 📚 Biblioteca Virtual — Lumina

Aplicación web de biblioteca virtual con panel de administración, catálogo de libros y asistente de IA.

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador
2. **Admin:** `admin@lumina.com` / `admin123`
3. O regístrate como miembro

## 📁 Estructura

```
biblioteca-lumina/
│
├── index.html        ← Aplicación completa
└── README.md         ← Este archivo
```

## 🌐 Publicar en GitHub Pages

1. Sube esta carpeta a un repositorio de GitHub
2. Ve a Settings → Pages → Branch: main → Save
3. Tu URL será: `https://tu-usuario.github.io/biblioteca-lumina/`

## ✨ Funciones

- Login y registro de usuarios
- Panel de administrador (agregar/eliminar libros)
- Catálogo con búsqueda y filtros
- Lector de libros a pantalla completa
- Asistente IA integrado (requiere API key de Anthropic)
